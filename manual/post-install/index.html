Automatic First-Run Tasks

On login, Hyprland autostart launches leenium-cmd-first-run once when ~/.local/state/leenium/first-run.mode exists.

  • Power profile and battery monitor timer setup.
  • UFW default policy + Docker integration + enable service.
  • /etc/resolv.conf symlink to systemd-resolved stub resolver.
  • GNOME interface defaults and icon cache refresh.
  • Elephant user service enable/start.
  • Cleanup of temporary installer sudoers files.
  • Critical notifications for keybindings and update/Wi-Fi.

Recommended Immediate Actions

  1. Connect networking if needed: leenium-launch-wifi
  2. Run system update: leenium-update
  3. Open keybindings cheatsheet: Super + K
  4. Check theme/font state: leenium-theme-list, leenium-font-list
